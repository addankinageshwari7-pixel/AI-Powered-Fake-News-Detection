About

A production-grade fake news classifier.

VeritasAI is a full-stack reference implementation of an AI-powered fake news detection system. It pairs a robust NLP pipeline with an interpretable linear model, then layers a clean, professional interface for analysts, researchers, and curious readers.

  • Detect REAL vs FAKE with calibrated confidence
  • Explain every verdict with contributing signals
  • Quantify trust, risk, and authenticity in one view
  • Stream live analytics through an integrated dashboard
Technology Stack
Backend
  • Python
  • Flask
  • Scikit-learn
  • Pandas / NumPy
  • Joblib
Frontend
  • HTML5 / CSS3
  • Bootstrap 5
  • Vanilla JS
  • Chart.js
ML Pipeline

Text cleaning → stop-word removal → TF-IDF (1–2 grams) → Logistic Regression (primary), Passive Aggressive Classifier (auxiliary). Models persisted with Joblib.
